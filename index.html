Demos

3D buttons

Bouttons are text-aligned. Button Default Button Grey Button Character icon Button Sprite icon

Code

// charset for text icons (source file UTF-8 encoded without BOM)
@charset "UTF-8";
@import url(button.less);
.button{
.button3d();
}
.button-grey{
.button3d( #bbb );
}
.button-ico{
.button3d( #E83A25, 2px, "?" );
}
.button-sprite{
.button3d( #93C28C, 20px, url([long sprite url]), -45px  3px );
}
