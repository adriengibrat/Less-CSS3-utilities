Less-CSS3-utilities

Demos

Simple 3D buttons

Button Default Bouttons are text-aligned.
Customize button appearance easily.
Button Round Use any markup you like.

Less CSS

@import url(button.less);
.button{
	.button3d();
}
.button-grey{
	.button3d( #bbb );
}
.button-round{
	.button3d( brown, 20px );
}

With image icons

Single image as icon
Even sprites
With same class to modify button

Less CSS

@import url(button.less);
.button-pic{
	.button3d( #93C28C, 20px, url(icon.gif) );
}
.button-sprite{
	.button3d( #93C28C, 10px, url(sprite.png), -45px  3px );
}

With counter icons

Button Counter Button Counter Button Counter Button Counter

Less CSS

@import url(button.less);
.button-counter{
	.button3d( #669199, 4px, counter );
}
.button-counter-alpha{
	.button3d( wheat, 0px, counter, lower-alpha );
}
// reset counter
.button-counter + .button-counter-alpha{
	counter-reset: button;
}
